# LITTLEMAKER CAMBODIA Website

A professional industrial and furniture website template for Cambodia.

## Updates
- **Shopping Cart**: Fully functional cart with **Quantity Controls** (+/-) and local storage persistence.
- **Invoice Generator**: New "View Invoice" button creates a clean, printable summary of the order.
- **Telegram Checkout**: Generates an invoice list with quantities and opens Telegram to send the order to sales.
- **Footer Navigation**: Contact buttons (Telegram, Facebook, Phone) have been moved to the footer for better layout.
- **Multi-page Structure**: Navigation now leads to separate HTML files for better SEO and organization.

## How to Run
1. Open `index.html` in your browser.
2. Navigate through the menu to view the shop and other pages.
3. Add products to the cart, adjust quantities, view the invoice, or click the cart icon to test the Telegram integration.
